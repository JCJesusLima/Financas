//4.6
Quem já esta utilizando a versão 4.5 deve apenas substituir os arquivos index.php, visa.php, master.php e login.php - substituir todo o conteúdo da pasta js no caminho /conf/js - e por fim adcionar a pasta dashboard na pasta principal do sistema para ter acesso a atualização para esta nova versão.

//4.5
A versão 4.5 está devidamente preparada para rodar em PHP7, visto que foi alterado a extensão mysql para mysqli, o que aumentou o desempenho do sistema além de permitir o update do servidor apache e php.

Para novos usuários do sistema:
Importe o arquivo lc-db-install.sql para que seja criado o banco de dados da aplicação.

Para usuários que ja utilizam o sistema:
Importe o arquivo lc-db-update.sql para preparar o banco de dados para receber a nova versão.
Obs.: O script prepara o banco da versão 4, 4.1, 4.2 e 4.3 para receber as versões 4.4 e 4.5. Caso utilizem
a versão 3.7 ou anterior, deverá fazer os devidos ajustes no banco de dados por conta própria.

E substitua todos os arquivos, inclusive o config.php no seu servidor.

De qalquer forma, lembre-se de realizar backup dos dados lançados no sistema, antes de ralizar a
atualização.
